# WML Legal — Locked Evidence Packet Format

This folder preserves the reusable WML Legal packet/email format that Ben approved after live visual QA.

## What this lock controls
- PDF evidence/issuer-review packet architecture.
- WML legal-team typography route.
- Evidence integrity rules: real source pixels for evidence; summaries must be labeled as summaries/indexes.
- Final outro rule: `Submission Packet Complete` appears once and only on the last page.
- WML outbound email footer: main-site logo only, clickable website, clickable email, no defensive disclaimer unless explicitly requested.

## Use this when
Ben says to use the saved WML format, legal-team packet style, intro/outro lock, WML footer, or research-saved packet format.
